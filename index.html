Oasis Palm Tree provides consultancy and support services for initiatives relating to Oasis LMF, the open source catastrophe loss modelling framework. We can offer advice, implementation services and on-going support for projects involving Oasis LMF, and can provide you with the latest solutions in the cat modelling and technology sectors.

The Oasis Loss Modelling Framework provides an open source platform for developing, deploying and executing catastrophe models. It uses a full simulation engine and imposes no restrictions on the modelling approach. Models are packaged in a standard format and available from many sources including commercial model vendors, academics and research groups.
